3. Datenerfassung auf dieser Website

a) Server-Log-Dateien durch den Hoster (GitHub Pages)

Diese Webseite wird auf GitHub Pages gehostet. Bei jedem Zugriff auf die Seite erfasst der Hoster (GitHub Inc., USA) automatisch Informationen, die Ihr Browser übermittelt. Dies sind typischerweise:

  • Browsertyp und Browserversion

Diese Daten werden von GitHub zur Sicherstellung des Betriebs und zur Sicherheitsanalyse verarbeitet. Ich als Betreiber dieser Seite habe keinen direkten Zugriff auf die vollständigen, unanonymisierten Log-Dateien.
